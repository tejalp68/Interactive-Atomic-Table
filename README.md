# 🧪 Interactive Periodic Table

An interactive web-based **Periodic Table of Elements** built using **HTML, CSS, and JavaScript**.
Users can click on any element to view detailed information such as atomic number, atomic mass, electron configuration, uses, and interesting facts.

---

## 📌 Features

* 📖 Displays all 118 chemical elements
* 🖱️ Click on any element to view detailed information
* 🎨 Responsive grid layout using CSS Grid
* ✨ Smooth hover effects and interactive UI
* ❌ Close button to hide element information panel
* 📚 Educational and beginner-friendly project

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout (Grid + Hover effects)
* **JavaScript (Vanilla JS)** – Dynamic element rendering and interactivity

---

## 📂 Project Structure

```
Interactive-Periodic-Table/
│
├── index.html      # Main HTML structure
├── style.css       # Styling and layout design
├── script.js       # JavaScript logic and element data
└── README.md       # Project documentation
```

---

## 📄 File Description

### 1️⃣ index.html

* Contains the webpage structure.
* Includes:

  * Title heading
  * Grid container for periodic table
  * Hidden information panel
  * Links to CSS and JavaScript files

---

### 2️⃣ style.css

* Designs the periodic table layout using **CSS Grid**
* Adds:

  * Hover animation
  * Responsive alignment
  * Styled information box
  * Close button styling

---

### 3️⃣ script.js

* Stores data of all 118 elements in an array of objects.
* Dynamically generates element tiles.
* Handles:

  * Click events
  * Displaying element details
  * Showing and hiding information panel

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`
4. The interactive periodic table will open in your browser

No installation or external libraries required ✅

# Interactive Atomic Table
This is my first ever web development project using HTML , CSS and JavaScript.it visually displays elements of the periodic table in a responsive and user friendly layout.
The project helped me understand DOM manipulation,layout structuring,an interactive UI components.

**FEATURES

-Color coded elements based on chemical categories
-Hover effects to highlight element details
-Responsive design for different screen sizes
-Element details pop-up

**TECHNOLOGIES USED

-HTML
-CSS
-JAVASCRIPT
